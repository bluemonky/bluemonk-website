# References (参考資料)

このフォルダはサイトに直接公開しない**参考用の資料**を置く場所です。

## 配下の目的

- `images/` — デザインの参考画像、ムードボード、ライセンス不明の素材、ペライチから引用した画像など、**実サイトには使わない** 画像を格納
- その他のファイル — 競合サイトのスクリーンショット、アイデアメモ、素材選定の候補集 など

## ⚠️ 注意

- このフォルダは `public/` の外にあるため、Next.js からは配信されません (サイト訪問者は見られません)
- Git には含まれますが、デプロイ後の Vercel 上でも公開されません
- 本番で使う画像は `public/images/` 配下に置いてください
